## TEE VM image downloads

TEE virtual machine images can be downloaded from **[downloads.buildernet.org](https://downloads.buildernet.org)**

<hr class="clean" />

## Allowed TEE measurements

Nodes are required to run a specific boot chain, codebase and configuration, verified using [TEE attestation and measurement mechanisms](verifiable-system-integrity).

:::info

BuilderNet TEE measurements are published at **[measurements.buildernet.org](https://measurements.buildernet.org)**

:::
